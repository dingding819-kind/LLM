【範圍】集合
【題目】
（）1、若集合A = {x | x² - 5x + 6 = 0}，集合B = {x | 1 < x < 4}，則A∩B = ？
(A) {2}
(B) {2, 3}
(C) {3}
(D) ∅

【答案】
（B）1、集合的交集問題
首先求集合A：
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
所以A = {2, 3}

集合B = {x | 1 < x < 4}（開區間，不含1和4）

A∩B表示既在A中又在B中的元素：
- 2∈A，且1 < 2 < 4，所以2∈(A∩B) ✓
- 3∈A，且1 < 3 < 4，所以3∈(A∩B) ✓

因此A∩B = {2, 3}

【範圍】函數（複合函數）
【題目】
（）2、若函數f(x) = 2x + 1，g(x) = x²，求(f∘g)(x) = ？
(A) 2x² + 1
(B) (2x + 1)²
(C) 2x² + 2x + 1
(D) 4x + 1

【答案】
（A）2、複合函數的計算
(f∘g)(x) = f(g(x))
意思是先計算g(x)，再將結果代入f函數

Step 1: 計算g(x) = x²
Step 2: 代入f函數：f(g(x)) = f(x²)
Step 3: 計算f(x²)：f(x²) = 2(x²) + 1 = 2x² + 1 ✓

(A) ✓ 正確答案
(B) 這是(f(x))²的形式
(C) 這是g(f(x))的計算結果
(D) 這是2g(x) + 1的形式

【範圍】三角函數
【題目】
（）3、在直角三角形ABC中，∠C = 90°，BC = 3，AC = 4，求sin A = ？
(A) 3/5
(B) 4/5
(C) 3/4
(D) 4/3

【答案】
（A）3、三角函數的定義
在直角三角形中：
- sin A = 對邊/斜邊
- cos A = 鄰邊/斜邊
- tan A = 對邊/鄰邊

已知：∠C = 90°（直角），BC = 3，AC = 4

Step 1: 計算斜邊AB
AB² = AC² + BC² = 4² + 3² = 16 + 9 = 25
AB = 5

Step 2: 角A的對邊是BC = 3
sin A = BC/AB = 3/5 ✓
